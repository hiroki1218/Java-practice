# Java基礎学習3

## プリミティブ型と参照型の違い(値vsアドレス値)

　**変数**は大きく以下の2つに分かれている。
違いとしてはプリミティブ型は値そのもの(リテラル)が格納されるのに対して、
参照型はメモリのアドレス値が格納される

　- プリミティブ型　(char, short, byte, int, boolean等)
　- 参照型(String等)

## `==` と　`equals`の違い

等価演算子(==)は以下のように動作する
- プリミティブ型：値そのもを比較
- 参照型：アドレス(参照先)を比較
  
`equals`メソッドは文字列同士を比較する際に使用される。
参照値は見ておらず文字列そのものを比較して結果を返す。

## String Poolとは

Javaのヒープメモリ内にある特別な領域である。
同じ内容のStringリテラルを共有・再利用することでメモリ使用量を最適化する仕組み。

補足
ヒープとはプログラムの実行中に動的にメモリを確保・解放できる**メモリ領域(ヒープ領域)**
